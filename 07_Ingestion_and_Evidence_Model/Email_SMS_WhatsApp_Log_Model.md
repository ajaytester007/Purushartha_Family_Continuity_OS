# Correspondence Ingestion Model

Normalize permitted correspondence into:
event_id, source, timestamp, participants, thread, content reference, consent class, fact/inference labels, sentiment hypothesis, topic, relationship stage, linked incidents.

## Analysis
Turn-taking, unanswered requests, escalation, repair attempts, contradiction, topic recurrence, response latency only when contextually meaningful.

Do not infer affection or guilt from response speed alone.
