# Release Notes - v2.1 Streamlit Workbench Reports Cockpit

## Purpose

v2.1 makes generated workbench reports easier to discover, browse, and publish by adding report manifest, report index, report cockpit documentation, and public report cards.

## Added

- Report manifest CSV.
- Report index Markdown.
- Reports Cockpit model.
- Streamlit report workbench upgrade plan.
- Pages report cards section.
- Wiki publishing update.
- v2.2 backlog for report-builder UI.

## Why This Matters

v2.0 generated reports, but the visibility path was scattered across local Markdown files, GitHub repository folders, and flattened Wiki pages. v2.1 introduces a consistent report catalog.
