# Reports Cockpit

## Purpose

The Reports Cockpit is the front door for generated workbench outputs.

## Report Families

1. Workbench Summary
2. Family Continuity Summary
3. Event Graph Report
4. Relationship Health Report
5. Burden Skew Audit
6. SCD2 State Timeline
7. Consent and Governance Report
8. Case Engine Output
9. Repair Plan Output
10. Retest Schedule

## Cockpit Columns

- Report Name
- Report Type
- Location
- Public or Private
- Generated By
- Last Updated
- Intended Audience
- Guardrail Status

## Operating Principle

Reports should help organize repair, governance, and continuity. They should not become accusations without human review and context.
