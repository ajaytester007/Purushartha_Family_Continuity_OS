# Daily Log Persistence

## Purpose

Store daily companion entries so the relationship can be reviewed over time.

## Fields

- log_id
- created_at
- mood
- relationship_energy
- partner_effort
- self_effort
- unresolved
- special_encounter
- gratitude_note
- concern_note
- daily_nudge

## Guardrail

Public deployment should use synthetic or sanitized entries only.
