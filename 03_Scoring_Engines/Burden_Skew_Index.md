# Burden Skew Index

Track time, mental load, emotional labor, money, caregiving, household work, and opportunity sacrifice.

For each domain:
- expected share;
- actual share;
- voluntariness;
- duration;
- recognition;
- recovery opportunity.

Temporary 80/20 can be healthy. Chronic unchosen 80/20 with entitlement is not.

Flag:
- invisible labor;
- obligation creep;
- one-way flexibility;
- sacrifice without restoration.
